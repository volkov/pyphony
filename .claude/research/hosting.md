## Бюджетные VPS: Hetzner, Vultr, DO и Linode

Четыре ключевых VPS-провайдера предлагают зрелые Terraform-провайдеры (все официальные, Partner-tier), нативную поддержку cloud-init и понятные REST API. Различия — в ценах, экосистеме и подходе к burst.

**Hetzner Cloud** — абсолютный лидер по цена/ресурсы. Линейка CAX (ARM Ampere): **€3.79/мес за 2 vCPU + 4 ГБ RAM + 40 ГБ NVMe**. Линейка CX (x86): €3.79/мес за аналогичные характеристики. Включённый трафик — **20 ТБ** в EU-регионах (Нюрнберг, Фалькенштайн, Хельсинки), что в 5–20 раз больше конкурентов. Terraform-провайдер `hetznercloud/hcloud` покрывает серверы, сети, файрволы, балансировщики, volumes. CLI `hcloud` спроектирован по образцу kubectl. Официальные SDK на Go и Python. Минусы: нет формальной системы burst CPU (мягкий shared), нет managed-баз данных, нет Kubernetes-сервиса. **⚠️ С апреля 2026 цены вырастут на ~30–40%** — CX22 станет ~€4.99/мес, но даже после повышения Hetzner остаётся самым дешёвым.

**Vultr** — самый дешёвый входной порог: **$2.50/мес** за 1 vCPU + 512 МБ + 10 ГБ SSD. High Performance на AMD EPYC + NVMe: $6/мес за 1 vCPU + 1 ГБ. Terraform-провайдер `vultr/vultr` зрелый, есть официальная документация по связке Terraform + cloud-init. **Больше всего EU-локаций** среди всех провайдеров: Амстердам, Лондон, Париж, Франкфурт, Мадрид, Варшава, Стокгольм, Манчестер. Managed DB (PostgreSQL, MySQL) от $15/мес, бесплатный control plane Kubernetes. Burst — мягкий shared, без формальной системы кредитов.

Managed PostgreSQL от $15/мес, бесплатный control plane K8s, App Platform от $5/мес. Бесплатные $200 кредитов на 60 дней. SLA **99.99%**. Burst — неявный shared vCPU, без кредитов.

## Большие облака: настоящий burst и бесплатные тарифы

Именно здесь находятся провайдеры с **формальными системами burstable CPU** — прозрачными кредитами, baseline-процентами и документированными лимитами.

### AWS EC2 T4g — самый прозрачный burst

Инстансы **t4g (ARM Graviton2)** — самый дешёвый burstable вариант на AWS. Ценообразование (us-east-1):

| Инстанс | vCPU | RAM | Baseline CPU | ~$/мес | Макс. burst на 100% |
|---|---|---|---|---|---|
| t4g.nano | 2 | 0.5 ГБ | **5%** | ~$3.07 | ~72 мин |
| t4g.micro | 2 | 1 ГБ | **10%** | ~$6.13 | ~144 мин |
| t4g.small | 2 | 2 ГБ | **20%** | ~$12.26 | ~288 мин |

Кредиты накапливаются ежечасно (6/12/24 кредитов/час), 1 кредит = 1 vCPU × 100% × 1 минута. По умолчанию включён **Unlimited mode**: при исчерпании кредитов вы продолжаете burst, но платите $0.04/vCPU-час сверх базовой линии. В Standard mode — жёсткое троттлинг до baseline.

**⚠️ Критически важный скрытый расход**: с февраля 2024 публичный IPv4 стоит **$3.60/мес**. Реальная стоимость t4g.nano: $3.07 + $3.60 (IP) + $1.60 (20 ГБ gp3) = **~$8.27/мес минимум**. Это делает дешёвые EC2-инстансы значительно дороже, чем кажется.

**AWS Lightsail** — упрощённая альтернатива: **$3.50/мес** (IPv6-only) или $5/мес (с IPv4) за 512 МБ + 2 vCPU + 20 ГБ SSD + 1 ТБ трафика. Всё включено, без сюрпризов. Burst — скрытый shared, без кредитной системы. 3 месяца бесплатно на выбранных планах.

### Azure B-series — burst с жёстким троттлингом

| Размер | vCPU | RAM | Baseline | ~$/мес | Макс. burst |
|---|---|---|---|---|---|
| B1ls | 1 | 0.5 ГБ | **5%** | ~$3.80 | ~76 мин |
| B1s | 1 | 1 ГБ | **10%** | ~$7.59 | ~160 мин |
| B1ms | 1 | 2 ГБ | **20%** | ~$15.11 | ~288 мин |

**Ключевое отличие от AWS**: в Azure **нет Unlimited mode**. При исчерпании кредитов — жёсткий троттлинг до baseline без возможности доплатить. Начальные 30 кредитов при развёртывании. 12 месяцев B1S бесплатно + $200 кредитов на 30 дней. Terraform `azurerm` — зрелый провайдер, cloud-init через `custom_data`.

### Google Cloud — лучший бесплатный вариант (в США)

**e2-micro** (2 shared vCPU, 1 ГБ RAM) — **бесплатно навсегда** в регионах us-west1, us-central1, us-east1. Включает 30 ГБ HDD, внешний IP, 1 ГБ egress. Burst — мягкий shared (~25% baseline). **Cloud Run** бесплатно: 180K vCPU-сек + 360K GiB-сек/мес, доступен в EU-регионах. $300 кредитов на 90 дней для новых аккаунтов. Для EU-размещения бесплатный tier не работает — e2-micro обойдётся в ~$6/мес. Terraform `google` — очень зрелый.

### Oracle Cloud — щедро, но сложно

**Бесплатно навсегда**: до **4 OCPU ARM (Ampere A1) + 24 ГБ RAM** + 2× AMD micro (1/8 OCPU, 1 ГБ RAM каждый) + 200 ГБ block storage + 10 ТБ egress + 2 автономные БД. По характеристикам — вне конкуренции.

Однако **серьёзные проблемы**: частые ошибки «Out of Capacity» при создании ARM-инстансов (иногда нужно пробовать днями), **рекламация idle-инстансов** (CPU < 20% за 7 дней → удаление), сложность IaC-настройки (RSA-ключи, tenancy/user OCID, compartment, VCN + subnet + internet gateway + route table + security list для одной VM). Terraform-провайдер `oracle/oci` зрелый, но **крайне verbose**. Не burstable — ARM даёт выделенные ядра.

**Рекомендация**: Oracle Cloud стоит использовать для «поиграться бесплатно», но не как основную платформу для IaC-управляемого проекта из-за сложности и ненадёжности capacity.

## PaaS и контейнерные платформы: Fly.io, Railway, Render

Эти платформы предлагают иную модель: вместо управления VM — деплой контейнеров с автоматическим масштабированием.

**Fly.io** — лучший для scale-to-zero контейнеров. Shared-cpu-1x + 256 МБ: **~$1.94/мес** (always-on) или центы при auto-stop. Выделенный IPv4: +$2/мес. Scale-to-zero через Fly Proxy: машина останавливается при простое, запускается при запросе — стоимость storage $0.15/ГБ/мес. 30+ регионов (10+ в EU). **⚠️ Terraform-провайдер официально deprecated** — Fly.io сознательно отказались от него, рекомендуя `flyctl` + CI/CD или Machines REST API. Для IaC-ориентированного инженера это серьёзный минус. `fly.toml` — декларативный конфиг приложения, но не полноценный IaC.

**Railway** — лучший Developer Experience. Hobby-план: **$5/мес** (включая $5 usage-кредитов). Подключаете GitHub → push → URL за 2 минуты. Автодетекция фреймворков через Nixpacks. One-click БД (PostgreSQL, MySQL, Redis). Тарификация по реальному потреблению CPU/RAM. Terraform-провайдер — **только community**, с пробелами (нет поддержки environments). Нет scale-to-zero. Egress дорогой: $0.10/ГБ.

**Render** — предсказуемые расходы + IaC. Starter: **$7/мес** (0.5 CPU, 512 МБ). Free tier: бесплатные static-сайты навсегда + web-сервис (0.1 CPU, 512 МБ) с cold start после 15 мин простоя. **Официальный Terraform-провайдер** `render-oss/render` (GA) + собственный IaC-формат **render.yaml** (Blueprints). Managed PostgreSQL от $6/мес. EU-регион: Франкфурт.

**Дополнительные варианты**:
- **Cloudflare Workers** — serverless на 300+ edge-точках, **$0/мес** (100K запросов/день), $5/мес (10M запросов), нулевой egress, D1/KV/R2 storage. Официальный Terraform-провайдер. Ограничение: V8-изоляты, не Docker.
- **Scaleway** — EU-нативный (Париж, Амстердам, Варшава). VPS DEV1-S: ~€4.76/мес. Serverless Containers с scale-to-zero. Официальный Terraform-провайдер.
- **Coolify** (self-hosted PaaS) — бесплатный open-source аналог Heroku. Установите на Hetzner VPS за €4/мес и получите полноценный PaaS с 280+ one-click сервисами, auto-SSL, git-деплоем.

## Зрелость IaC-экосистемы: кто где

Для AI/ML-инженера, привыкшего описывать инфраструктуру кодом, IaC-поддержка — ключевой фактор выбора. Вот реальная картина:

**Tier 1 — отличный IaC** (официальные Terraform + Pulumi провайдеры, cloud-init, качественные API и CLI):
- Hetzner Cloud (`hetznercloud/hcloud`, Pulumi `@pulumi/hcloud`, CLI `hcloud`, SDK Go/Python)
- DigitalOcean (`digitalocean/digitalocean`, Pulumi `@pulumi/digitalocean`, CLI `doctl`)
- Vultr (`vultr/vultr`, Pulumi `@ediri/vultr`, CLI `vultr-cli`)
- Linode/Akamai (`linode/linode`, Pulumi `@pulumi/linode`, CLI `linode-cli`)

**Tier 2 — хороший IaC, но избыточная сложность для хобби**:
- AWS (gold-standard Terraform `hashicorp/aws`, но VPC/subnet/SG boilerplate для EC2; Lightsail проще)
- Google Cloud (`hashicorp/google`, но project/IAM/billing setup)
- Azure (`hashicorp/azurerm`, но resource group/VNet/NSG/NIC overhead)
- Scaleway (`scaleway/scaleway`, хороший EU-вариант)
- Render (`render-oss/render` GA + render.yaml Blueprints)

**Tier 3 — проблемный IaC**:
- Oracle Cloud (`oracle/oci` — зрелый, но требует RSA-ключи, 4+ OCID, VCN-стек для одной VM)
- **Fly.io — Terraform-провайдер deprecated** (сознательно); только `flyctl` + Machines API
- Railway — только community Terraform-провайдер, неполный, с rate-limiting багами

**Дополнительные IaC-инструменты**, совместимые со всеми VPS-провайдерами: **Kamal** (zero-downtime Docker-деплой через SSH), **Ansible** (конфигурация серверов), **NixOS** (полная декларативная система), **Docker Compose** (простейший вариант). Оптимальный стек: **Terraform → cloud-init → Docker Compose/Kamal**.

## Практические рекомендации для pet-проекта

Исходя из профиля «AI/ML-инженер, важен IaC, бюджет до $20/мес, хобби-проект»:

**🥇 Лучший выбор: Hetzner Cloud CAX11 (ARM) — €3.79/мес**
Идеальное сочетание: 2 vCPU ARM + 4 ГБ RAM + 40 ГБ NVMe + 20 ТБ трафика. Отличный Terraform-провайдер, cloud-init, чистый API. Для ML-инференса на ARM (ONNX Runtime, llama.cpp) — хватает. На оставшийся бюджет (~$15) можно добавить managed PostgreSQL на DO/Linode ($15/мес) или второй сервер. Комбо **Hetzner + Coolify** даёт полноценный PaaS за ~€5–8/мес.

**🥈 Лучший DX: DigitalOcean $6/мес + App Platform**
Если важнее экосистема, документация и managed-сервисы. $6 Droplet + $5 App Platform + $5 Spaces = $16/мес с полным стеком. Или DOKS (бесплатный control plane) + $6 worker = мини-Kubernetes за $6/мес.

**🥉 Лучший бесплатный: Oracle Cloud ARM (4 OCPU + 24 ГБ)**
Если готовы мириться со сложностью OCI Terraform, проблемами capacity и риском рекламации idle-инстансов. Реально полезно для тестирования ML-моделей, где нужно много RAM. Совет: перейдите на Pay-As-You-Go аккаунт (улучшает доступ к capacity, бесплатные ресурсы сохраняются).

**🏅 Лучший burst: AWS t4g.micro — ~$6.13/мес (+$3.60 IPv4)**
Единственный провайдер с полностью прозрачной кредитной системой. Baseline 10%, burst до 100% на ~144 минуты. Unlimited mode позволяет не бояться троттлинга. Но реальная стоимость с IPv4 и EBS: ~$11–12/мес за 1 ГБ RAM. Имеет смысл, если вы уже в экосистеме AWS.

**🏅 Лучший scale-to-zero: Fly.io ~$2–5/мес**
Для нечасто используемых API/сервисов. Машина спит, пока нет запросов → платите копейки. Но отсутствие Terraform — серьёзный минус. Альтернатива: **Cloudflare Workers** ($0–5/мес) для лёгких serverless-функций с полным Terraform-провайдером.

## Заключение: матрица решений

Универсального «лучшего» провайдера нет — выбор зависит от приоритетов. Для **максимума ресурсов за минимум денег с хорошим IaC** — Hetzner. Для **прозрачного burst CPU** — AWS T4g или Azure B-series (но с overhead по стоимости и сложности). Для **быстрого деплоя без серверов** — Railway или Render. Для **бесплатного эксперимента** — Oracle Cloud (с оговорками) или GCP e2-micro (только US).

Ключевой инсайт: бюджетные VPS-провайдеры (Hetzner, Vultr, DO, Linode) за $4–6/мес дают **в 2–4 раза больше ресурсов**, чем аналогичные по цене инстансы у гиперскейлеров, при сопоставимом качестве IaC-инструментов. Гиперскейлеры оправданы, когда критичен формальный burst, масштабный free tier или специфические managed-сервисы. Для типичного pet-проекта AI/ML-инженера оптимальная стратегия: **Hetzner ARM + Terraform + cloud-init + Docker Compose** — и весь бюджет уложится в €4–8/мес с запасом на эксперименты.
